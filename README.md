# Neon Live Streaming Platform

Neon is a Microservice Live Streaming Platform for Software Engineering Streamers.

## Technologies Used

- Elixir
- Phoenix
- Membrane
- SvelteKit
- Kong
- Docker
- Docker Compose
- Postgresql
